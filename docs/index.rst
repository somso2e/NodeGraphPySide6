NodeGraphPySide6 |version_str|
#########################

.. image:: _images/logo.png
    :align: center

NodeGraphPySide6 a node graph UI framework written in python that can be implemented
and re-purposed into applications supporting PySide2.

.. image:: _images/screenshot.png

Install
*******

NodeGraphPySide6 is available from the `The Python Package Index (PyPI) <https://pypi.org/project/NodeGraphPySide6/>`_ so
you can install via ``pip``.

.. code-block::

    pip install NodeGraphPySide6

or alternatively you can download the source `here <https://github.com/jchanvfx/NodeGraphPySide6/archive/refs/heads/main.zip>`_.

Getting Started
***************

To get started see the `basic_example.py <https://github.com/jchanvfx/NodeGraphPySide6/blob/main/examples/basic_example.py>`_
script or checkout the :ref:`General Overview` section.
